2.extern是只声明不定义(此时不能跟着值,不然失效)，从而让别的文件的变量能为程序所知，提示编译器遇到此变量和函数时在其他模块中寻找其定义；而不带extern的是声明并定义，会获得一个具名的可操作的对象实体。
